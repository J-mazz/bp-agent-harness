# Findings (git-ignored)

Evidence, notes, and draft reports live here, organized as
`findings/<program>/<asset>/`. Everything under this directory except this README is
**git-ignored** because it may contain confidential target data.

Handling rules (see `../context/rules-of-engagement.md`):
- Redact secrets and PII; keep only the minimum needed to prove an issue.
- Never commit, publish, or share these files.
- Disclose findings only through the program's HackerOne page.
